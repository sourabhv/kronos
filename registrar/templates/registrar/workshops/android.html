Linux

You guys are incredible if you work on Linux machines. Hi-5 on that already.

Since you are on an incredible machine you don't you figure out how to install yourself? Just grab the latest build file from here.

For the newbies, welcome and please learn how to do this process since you'll be doing it often in your life. Your downloaded file should look like this Android-Studio-Bundle-135.1245622-linux.tgz. Now simply do the following(make sure you're in the correct directory):

  • $ man tar #read about tarball extractions
  • $ mkdir ~/bin/
  • $ mv Android-Studio-Bundle-135.1245622-linux.tgz ~/bin/
  • $ cd ~/bin/
  • $ tar zxvf Android-Studio-Bundle-135.1245622-linux.tgz
  • #this might take a while.
  • $ cd android-studio

Now this android-studio folder is where all the files are. To run android-studio from your terminal. We need to do 2 things: 1) make it visible in our path 2) create symbolic link in /usr/bin to it

  • $ PATH=$PATH:$HOME/bin/android-studio/bin/studio.sh
  • $ sudo ln -s ~/bin/android-studio/bin/studio.sh /usr/bin/studio

And that's it! Don't you feel like a hacker already? :)
